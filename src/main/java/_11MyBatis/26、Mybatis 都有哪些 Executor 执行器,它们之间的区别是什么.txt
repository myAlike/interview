26、Mybatis 都有哪些 Executor 执行器？它们之间的区别是什么？
答：Mybatis 有三种基本的 Executor 执行器，SimpleExecutor、ReuseExecutor、BatchExecutor。 1）SimpleExecutor：每执行一次 update 或 select，就开启一个 Statement 对象，用完立刻关 闭 Statement 对象。2）ReuseExecutor：执行 update 或 select，以 sql 作为 key 查找 Statement 对象，存在就使用，不存在就创建，用完后，不关闭 Statement 对象，而是放置于 Map3） BatchExecutor：完成批处理。
